Book Library App
Live Demo: https://peimanhp.github.io/Library/

Book Library App is a user-friendly web application built with pure JavaScript and Bootstrap. This app allows users to easily keep track of their book collection, including the title, author, number of pages, and whether they have read the book or not.

With its smooth transitions and responsive design, Book Library App provides a seamless experience across different devices. The app ensures that users can access their library anytime, anywhere.

One of the key features of this app is the modal window, which provides detailed information about each book in a visually appealing manner. Users can quickly view additional details such as synopsis, publication date, and genre by simply clicking on a book.

Whether you are an avid reader or just starting your book collection, Book Library App is the perfect tool to organize and manage your books efficiently. Try it out now and experience the convenience of having all your books at your fingertips!
